# GBA PROJECT

A low-level game development journey

## Introduction

This is a Game Boy Advance game currently under development by three Computer Science students from **UFRN - Brazil**.

## Proposal

The primary goal of this project is to explore low-level programming concepts, memory mapping, and hardware constraints by developing software for a 
32-bit RISC architecture (ARM7TDMI).

## Technologies & Tools

* **Language**: C
* **Toolchain**: devkitPro
* **Debug & Test**: mGba Emulator

## How to build

1. Install devkitPro
2. run the command 'make' in the root directory
3. load the output file .gba in an emulator of your choice
4. have fun 😄
